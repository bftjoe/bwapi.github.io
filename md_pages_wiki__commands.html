Commands

This page contains a list of /commands specified and reserved by BWAPI.

Release Commands

This list contains commands that are available in both RELEASE and DEBUG modes.

Command Description
/leave Leaves the game.
/speed # Sets the game speed to the number specified, 0 being fastest. Blank will reset the speed.
/fs # Sets the frame skip to the number specified. Blank will reset the frame skip value to 1.
/cheats Enters several cheat codes.
/restart Restarts the game.
/nogui Toggles the display of Starcraft's Graphical User Interface (GUI).
/wmode Toggles the display between windowed mode and fullscreen mode.
/grid Toggles the display of the tile grid.
/record Begins video recording. Should only use in BWAPI windowed mode.
/stoprecord Stops video recording.

Debug Commands

This list contains commands that are only available in DEBUG mode.

Command Description
/latency A test that displays results from the latency commands.
/dlgdebug A test for the BW::Dialog class usage.
/pathdebug Toggles debugging of BW-level pathfinding.
/unitdebug Toggles debugging of BW-level unit data.
/test Used for quickly testing various temporary things.
/resize Sets the unfinished resolution hack display resolution to 1024x768.
/hud Toggles the display of the HUD (minimap/buttons/stats GUI).

Hotkeys

Hotkey Description
ALT + ENTER Toggles the display between windowed mode and fullscreen mode.
ALT When held down, it will prevent the cursor from leaving the game window. This is only available in Windowed Mode.
CTRL When held down while resizing the window in Windowed Mode, it will maintain its original aspect ratio of 4:3.

Cheat Codes

This list contains cheat codes that were already part of StarCraft. They can only be used in Single Player games. The 'player' refers to the only human player in single player games.

Command Description
power overwhelming All non-player attacks deal 0 damage.
show me the money Gives the player 10000 minerals and 10000 vespene gas.
operation cwal All units and structures build times are significantly reduced. Upgrade and research times become 0.
the gathering All abilities cost no energy.
game over man Ends the game with Defeat.
staying alive Prevents the game from ending.
there is no cow level Ends the game with Victory.
whats mine is mine Gives the player 500 minerals.
breathe deep Gives the player 500 gas.
something for nothing Increases the level of all upgrades for the player by 1.
black sheep wall Provides the player with full vision of the entire map.
medieval man All abilities are enabled, despite not being researched.
modify the phase variance Removes the technology tree restrictions on units and structures for all players.
war aint what it used to be Disables the fog of war.
food for thought All players can exceed their maximum supply.
ophelia Allows the player to start a specific campaign map immediately. After entering this command, type the campaign id, i.e. xprotoss1, terran2, etc.
radio free zerg Plays the hidden Zerg song. Must play as the Zerg in BroodWar.
